<h1 align="center">hackcqooc</h1>

一个重庆高校在线开放课程平台自动化工具包。

## [Hackcqooc 的使用样例](./sample.py)

在终端环境中分别设置变量 `USERS` 和 `PASSWORD`
后运行 `python3 sample.py` 它将会告诉你 Hackcqooc 的运行方式。

## 安装

```
pip3 install hackcqooc
```

## 功能

1. 🔑 使用帐号密码直接登录
2. ℹ️ 获取用户信息
3. 🏛️ 获取用户选课信息
4. 📚 获取用户课程详情
5. ⏳ 跳过对应课程
